> 플러터가 제공하는 키보드 유형

| 키보드 유형  |                용도                |
| :----------: | :--------------------------------: |
|     text     |            기본 텍스트             |
|  multiline   | 메모 같이 여러 줄을 입력할 때 사용 |
|    number    |            숫자 키보드             |
|    phone     |        전화번호 전용 키보드        |
|   datetime   |          날짜 입력 키보드          |
| emailAddress |         이메일 입력 키보드         |
|     url      |            주소 입력 창            |
